# kotlin-web-maven-spring-thyme-hello-world

## Description
A springboot web app with thymeleaf support.

## Tech stack
- kotlin
- maven

## Docker stack
- openjdk:11

## To run
`sudo ./install.sh -u`
Available at http://localhost

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
